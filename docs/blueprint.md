# **App Name**: Chromatic Tales

## Core Features:

- Image Capture: Web interface allowing users to upload images or use their webcam to capture the 3x3 color grid.
- Color Detection: AI-powered color detection to identify the color in each block of the 3x3 grid using basic RGB rules. A tool makes reasoning to accomplish it.
- Pattern Matching: Pattern matching logic to compare the detected color matrix against pre-defined templates.
- Action Mapper: Action mapper that triggers a video or animation based on the matched template, using locally stored videos.
- Matrix Display: Display the detected color matrix to the user.
- Fallback Message: Display a fallback message when no template matches the detected pattern.

## Style Guidelines:

- Primary color: Saturated blue (#4285F4), evoking reliability, focus and precision in color analysis. It contrasts well with both light and dark backgrounds.
- Background color: Light gray (#F5F5F5). It's easy on the eyes and provides a clean backdrop, to ensure readability.
- Accent color: A vibrant purple (#A28BFA). Its greater brightness and saturation are ideal for making interface elements pop.
- Font: 'Inter' sans-serif for a clean, modern look suitable for both the user interface and any displayed text.
- A clean, modular layout will present each stage (image input, color detection, video playback) in a clear sequence.
- Minimal, subtle transitions to confirm actions such as scanning and template matching.
